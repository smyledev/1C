# Условный оператор

```

ИначеЕсли <Условие> Тогда

Иначе

```

## Краткая форма
```
?(<Условие>, <ВыражениеЕслиУсловиеИстина>, <ВыражениеЕслиУсловиеЛожь>)
```


## Приоритеты проверки условий
1. ()
2. НЕ
3. И
4. ИЛИ
