# Переменные и примитивные типы данных

## Объявление и инициализация переменных
```
Перем СозданнаяПеременная;
ЧисловаяПеременная = 10;
```

## Примитивные типы данных и применение стандартных функций по работе с ними

### Числа
```
ДробноеЧисло = 10.55;
ЦелаяЧасть = Цел(ДробноеЧисло); // 10

ЧислоПослеОкругления = Окр(ДробноеЧисло, 1); // 10.6
ЧислоПослеОкругления2 = Окр(ДробноеЧисло, 1, РежимОкругления.Окр15как10); // 10.5
```

### Даты
```
ВыбраннаяДата = ‘20230809’;

Месяц = Месяц(ВыбраннаяДата); // 8

НачалоМесяца = НачалоМесяца(ВыбраннаяДата ); // 01.08.2023 0:00:00

КонецДня = КонецДня(ВыбраннаяДата ); // 09.08.2023 23:59:59

НеделяГода = НеделяГода(ВыбраннаяДата ); // 33

ВыбраннаяДата = ДобавитьМесяц(ВыбраннаяДата , 3); // 09.11.2023 0:00:00

ТекущаяДата = ТекущаяДата(); // 03.04.2023 17:08:50

ЗавтрашнийДень = ТекущаяДата() + 60*60*24;
```

### Строки
```
// 1 вариант переноса строки
МногострочнаяСтрока = «Первая строка

|Вторая строка»;

// 2 вариант переноса строки
ПерваяСтрока = «1-я строка»;
ВтораяСтрока = «2-я строка»;
РезультатСоединенияСтрок = ПерваяСтрока + Символы.ПС + ВтораяСтрока;

// 1 вариант добавления кавычки в строку
НазваниеОрганизации = «ООО «»Техно»»»; // ООО «Техно»

// 2 вариант добавления кавычки в строку
КраткоеНазвание = «Техно»;
ПолноеНазвание = «ООО » + «»»» + КраткоеНазвание + «»»»; // ООО «Техно»

ТестоваяСтрока = » тест «;

ДлинаСтроки = СтрДлина(ТестоваяСтрока); // 10

СтрокаБезПробеловСлева = СокрЛ(ТестоваяСтрока); // «тест «
СтрокаБезПробеловСправа = СокрП(ТестоваяСтрока); // » тест»
СтрокаБезПробеловСлеваИСправа = СокрЛП(ТестоваяСтрока); // «тест»

ПятьСимполовСНачала = Лев(ТестоваяСтрока, 5); // » те»
ЧетыреСимволаСКонца = Прав(ТестоваяСтрока, 4); // «т «

ПодстрокаНачалоКонец = Сред(ТестоваяСтрока, 4, 4); // «тест»

ИндексНайденногоСимвола = СтрНайти(ТестоваяСтрока, «е»); // 5
ИндексПервогоСимволаВНайденнойПодстроке = СтрНайти(ТестоваяСтрока, «ст»); // 6

СтрокаВВерхнемРегистре = ВРег(ТестоваяСтрока); // » ТЕСТ «
СтрокаСПервымЗаглавнымСимволом = ТРег(ТестоваяСтрока); // » Тест «

СтрокаПослеЗамены = СтрЗаменить(ТестоваяСтрока, «е», «о»); // » тост «

СтрокаБезПробелов = СтрЗаменить(» те ст «, » «, «»); // «тест»

ЧислоСтрок = СтрЧислоСтрок(МногострочнаяСтрока); // 2

ВтораяСтрока = СтрПолучитьСтроку(МногострочнаяСтрока, 2); // «Вторая строка»
```

## Правило преобразования типов
> Приведение типа происходит относительно первого операнда


## Проверка типа
```

  Сообщить(«Переменная ТекДата эта дата»);
КонецЕсли;
```
